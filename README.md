# Config

after install dummysocks, you should configure it:

click *Edit Config* button:

```
......
<dict>
    <key>host</key>
    <string>diablo</string>
    <key>user</key>
    <string>huoxy</string>
    <key>private_key</key>
    <string>/Users/huoxy/.ssh/id_rsa</string>
</dict>
......
```

*NOTE: private_key MUST BE a absolute path.*
